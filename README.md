# Hotel Management System

## Overview
The Hotel Management System is a web-based application developed to automate and simplify hotel operations such as room booking, customer management, billing, payment processing, and checkout operations. The system provides an efficient way to manage customer records and hotel services through an intuitive user interface.

## Features
- Customer Registration and Management
- Room Booking System
- Customer Status Tracking
- Billing Calculation and Management
- Payment Processing
- Checkout Operations
- Admin Login Authentication
- Session-Based Access Control
- Input Validation
- Responsive User Interface

## Technologies Used
- Java Servlets
- JDBC
- MySQL
- HTML
- CSS
- Apache Tomcat

## Database
The application uses MySQL for storing:
- Customer Details
- Booking Information
- Billing Records
- Admin Credentials

## Authentication & Security
- Admin Login System
- Username and Password Authentication
- HTTP Session Management

- Secure Database Connectivity using JDBC

## Key Functionalities
### Room Booking
Allows administrators to register customer details and allocate rooms efficiently.

### Customer Management
Stores and manages customer information using a MySQL database.

### Billing System
Automatically calculates customer bills based on stay duration and services.

### Authentication
Implements an Admin Login System using Java Servlets and HTTP Sessions to secure hotel operations.

### Logout Functionality
Provides secure session invalidation and redirects users to the login page.
